# Create: Shield

**Create: Shield** 是一个 Minecraft NeoForge 1.21.1 模组，为 **Create** 模组添加两个动力方块。

## 前置

- **Minecraft** 1.21.1
- **NeoForge** 21.1+
- **Create** 6.0.10+
- **Sable** 1.2.2（可选，用于子空间推动）

## 方块

### 发条方块（Clockwork Block）— 动力电池

可以存储和释放 SU（应力单元）。

| 面 | 功能 |
|---|---|
| 背面 | 动力输入 |
| 正面 | 动力输出 |
| 顶面 | 滚轮调节功率（1–256 × 256 SU/rpm） |

- **充电模式**（无红石信号）：输入转速，以 `功率 / 速度` 消耗应力并存储能量
- **放电模式**（有红石信号）：输出转速，以存储的能量产生应力容量
- Shift + 右键切换输出转速（16/32/64/128 RPM）
- 能量在破坏/放置时保留

### 斥力盾（Shield Block）

以圆锥形范围推动实体和子空间。

| 面 | 功能 |
|---|---|
| 左侧 | 放入物品设置无线频率（两个槽位） |
| 右侧 | 右键打开 GUI 调节参数 |
| 正面 | 放置喷嘴（Nozzle）改变锥角范围 |
| 背面 | 动力输入 |

#### 参数

- **斥力角度**（φ）：45°–270°，放置喷嘴后可调，无喷嘴时固定 45°
- **斥力流量**：1–32，受无线红石和有线红石信号强度影响（= max(无线, 有线) × 2）
- **斥力范围**：1–32 格

#### 特性

- 圆锥扫描：仅推动面向方向锥角内的实体
- 射线检测：被方块遮挡的实体不会被推动，也不会产生粒子
- 频率共享：相同频率对的 Shield 共享扫描结果，避免重复推动
- 同盟子空间：相同频率的 Shield 所在子空间互不推动
- 推力衰减：`flow × (cos(distance/range + π/2) + 1)`，越远越弱
- 护盾边界粒子仅在有射线可达的球面（r=range）上生成

#### 子空间支持

- Shield 在 Create 子空间（Sub-Level）中可推动其他子空间
- 支持反冲力：射线击中方块时对自身子空间施加反作用力
- 装在动力装置（Contraption）上时也能推动实体

## 构建

```bash
./gradlew build
```

输出 JAR：`build/libs/create-shield-1.21.1-<version>-<revision>.jar`

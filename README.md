# Create: Shield

**Create: Shield** 是一个 Minecraft NeoForge 1.21.1 模组，为 [Create](https://github.com/Creators-of-Create/Create) 和[sable](https://github.com/ryanhcode/sable)(可选)模组添加两个动力方块：**发条方块**（动力电池）和 **斥力方块**（锥形实体推动器）。

## 前置依赖

| 依赖 | 版本 |
|------|------|
| Minecraft | 1.21.1 |
| NeoForge | 21.1+ |
| Create | 6.0.10+ |
| Sable | 1.2.2（可选，子空间物理推动） |
| Java | 21 |

## 一、发条方块（Clockwork Block）—— 动力电池

发条方块是一种可以**储存和释放旋转动能**的机械装置，相当于一个巨型"充电电池"。

### 1.1 方块结构

| 面 | 功能 |
| 背面 | 动力**输入**端，连接动力源（如电动机、水车）
| 正面 | 动力**输出**端，向机械网络供电 

### 1.2 工作模式

#### 充能模式（无红石信号）

- 从背面接收旋转动力，将动能转化为内部储存能量
- **应力消耗** = `转速 ×  256`（SU）
- 储能上限约 **20 亿 SU·tick**
- 充能时需要外部网络提供足够的应力容量

#### 放电模式（有红石信号）

- 释放储存的能量，从正面输出旋转动力
- **提供无限应力容量**（由储存的能量支撑）
- **放电消耗** = `ceil(应力消耗)` SU·tick，从储存能量中扣除
- 当储存能量耗尽时自动停止输出

## 二、斥力方块（Shield Block）—— 锥形实体推动器

斥力方块是一种**锥形范围实体推动装置**，利用旋转动力在前方产生斥力场，将实体推离。

### 2.1 方块结构

| 面 | 功能 |
| 背面 | 动力**输入**端 |
| 正面 | 斥力波发射方向，可安装**分散网（Nozzle）**调整范围 |
| 左面 | 右键打开 **GUI 参数调节面板** |
| 右面 | 两个**频率槽位**，放入物品设置无线频率 |

### 2.3 锥形扫描机制

斥力方块在前方形成一个**圆锥体扫描区域**：

- **斥力角度 φ** = 45° – 270° | 锥形范围的张开角度。需安装分散网后才可调，无分散网时固定 45° |
- **斥力范围 r** = RPM/8 |锥形范围的半径|
- **斥力强度 flow** = 红石信号强度
- 只有当实体/物理化结构（航空学）位于锥角内,且不被视线遮挡才会被推动
- 当喷嘴推动物理化结构/主世界时，自身会被反推
### 2.5 频率联动系统

斥力方块支持**无线频率联动**，实现多个方块协同工作：

1. 在两个频率槽位中放入物品设置频率
2. 支持 Create 无线红石调节
3. 拥有相同频率对的斥力方块的物理化结构互为「同盟」，所在的子空间不会互相推动，防止同频结构被吹走
